# UNO

Codigo que simula uma partida de UNO entre dois jogadores com 24 cartas. As cartas sendo de 0 a 5 e das cores: vermelho, azul e verde, mais 3 cartas +2 de cada cor e o mesmo para carta reverso.

- Nesse código é implementada a estrutura de dados Pilha, Listas Encadeadas com conceitos de recursividade, manipulação de String e Alocação Dinâmica em C.

![imagem-jogo](jogo.png)

## Como usar

O UNO funciona tanto no sistema Windows quanto em Linux.
- Linux pode-se usar compilador da linguagem C gcc em um IDE ou no terminal
- Windows qualquer compilador de C como [Dev-C++](https://www.dev-cpp.com/)
